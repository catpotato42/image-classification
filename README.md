# image-classification
sorts images into different classes using a ResNet with 18 layers.

# Setup instructions
Run:
`python -m venv env source`
`env/bin/activate` On Windows: `env\Scripts\activate`
`pip install -r requirements.txt`
Then, simply run the command `python3 .\output_to_image` from the root folder. The images should give you a hint as to what gesture you need to do in order for the model to recognize it. Attempt in a well lit room.
